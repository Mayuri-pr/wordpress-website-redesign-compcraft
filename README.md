# 🌐 WordPress Website Redesign (Compcraft)

---

## 📌 Project Overview
This project is a homepage redesign for a client website developed using WordPress. The objective was to modernize the existing design, improve user experience, and create a clean, visually appealing interface while maintaining brand consistency.

---

## 🚀 Features
- Homepage redesign with modern UI/UX  
- Fully responsive design across all devices  
- Improved layout and visual hierarchy  
- Clean and user-friendly interface  
- Enhanced readability and content structure  
- Optimized for better user engagement  

---

## 🛠️ Tech Stack
- WordPress
